Atividades

Agora que você já treinou a leitura, é hora de escrever. Crie um arquivo index.html, importe o CSS do Bootstrap via CDN e construa os três elementos abaixo usando apenas tags HTML básicas (<div>, <p>, <span>, <h1>) e as classes do Bootstrap

Desafio A: O Alerta de Sucesso

  • Crie uma caixa div.
  • O fundo deve ser verde de sucesso e o texto claro (light)
  • Deve ter um preenchimento interno (padding) de nível 3 em todos os lados.
  • O "Operação ralizada com sucesso!" deve estar centralizado em itálico.
  • Adicione cantos arredondados padrão.
Operação Realizada com sucesso!

Desafio B: O Crachá

  • Crie um texto em linha (span)
  • Aplique um fundo escuro e o texto claro
  • A fonte deve ser bem pequena (escala 6) e com peso em negrito.
  • Adicione o preenchimento interno horizontal (eixo x) de nível 2 e vertica (eixo y) de nível 1
  • Arredonde os cantos ao máximo em formato de pílula.
