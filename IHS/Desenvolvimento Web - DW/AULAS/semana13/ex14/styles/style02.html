CSS Interno

Um estilo interno é aqule que fica configurado dentro da área do <head> do seu documento.

A grande vantagem dessa maneira em relação do modelo inline é que podemos escrever uma configuração única para todos os elementos da mesma familia. Porem exemplo, todos os parágrados deste documento estão com a mesma configuração de cor, e eu só preciso escrever isso uma vez.

A grande desvantagem dete modelo em relação a ultima estrutura que veremos é que no CSS externo (a estrura do próximo exercicio) é que o estilo não pode ser reaproveitado por outra páginas.

IMPORTANTE: Você pode misturar CSS inline com CSS interno em um mesmo documento sem problemas! Vamos fazer um pequeno TESTE! É deste processo de aplicarmos os estilos em diferentes niveis que surge o conceito de "Folhas de Estilo em Cascata"
