Revisão HTML

O que é HTML?

O HTML (HyperTextoMarkupLanguage) é um uma linguagem de marcação em que te permite desenvolver sites, sendo ela a principal entre os desenvolvedores por sua versatilidade, simplicidade e compatibilidade.

Como desenvolver um site?

Aqui vai alguma recomendações o qual (em minha visão) você deveria aderir :D

Instalando uma IDE

A IDE é um espaço de desenvolvimento que para desenvolvedores Web não dá para se jogar fora! Lá onde é facilitada as etapas de produção do site.

Para marinheiros de primeira viagem, a IDE Visual Studio Code é uma excelente escolha!

Baixe VSC aqui

Extensões Úteis

As extensões são complementos que adicionam novas funções para você que é desenvolvedor!

  1. Live Server - Uma extensão que permite você colocar o seu arquivo HTML em um servidor local, permitindo atualizar mundaças em tempo real, um kit completo para colocar o seu site em teste!
  2. Open in Browser - Essa extensão funciona como a anterior, mas o diferencial dele é que é um atalho para abrir o seu arquivo no seu Browser de preferência. Facilitando muito o desenvolvimento, sem consumir muito!

Configurações recomendadas

Aqui onde você encontrará recomendações de Configurações que na minha opnião, deveria ser obrigatórias!

  1. Word Wrap - Uma configuração que facilita a visualização de blocos de textos extensos, ajudando a visualização em 100%!
  2. Auto Save - Chaga de "Ctrl + S" a cada linha! essa configuração salva a cada pausa que você está digitando. Sem preocupações e poupando o seu tempo!

Os seus parceiros de versionamento Git/GitHub

Quando você começa a codar, você percebe as inúmera mudanças que fará do mesmo arquivo. Para melhor controle e organização, você utilizará essas ferramentas para recuperar versões anteriores e manter o seu projeto nos trinques!

Git Vs GitHub
